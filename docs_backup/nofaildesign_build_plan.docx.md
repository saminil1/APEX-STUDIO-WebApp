# **2\. UX/UI 상세 분석**

## **2.1 스크롤 기반 스토리텔링 구조**

이 사이트의 핵심 UX 전략은 '스크롤 내러티브(Scroll Narrative)'입니다. 사용자가 스크롤할수록 마치 영업 프레젠테이션을 보는 것처럼 논리적으로 설득당하는 구조입니다.

**스크롤 설득 흐름도**

| 단계 | 목적 | 핵심 메시지 | 심리 효과 |
| :---- | :---- | :---- | :---- |
| ①  Hook | 관심 유발 | 떠나는 vs 머무는 홈페이지 | 문제 인식 |
| ②  Trust | 신뢰 구축 | 1,100+ 프로젝트, 인터뷰 영상 | 사회적 증거 |
| ③  Pain | 고통 확인 | 읽지 않고, 믿지 않고, 행동 안 함 | 공감 유발 |
| ④  Insight | 교육 | 매출 \= 유입 x 전환, 투자 개념 | 권위 형성 |
| ⑤  Solution | 해결책 제시 | 3가지 원칙 (읽게/믿게/행동하게) | 희망 제시 |
| ⑥  CTA | 전환 | 내 사업에 맞춘 전략 받기 | 행동 유도 |

## **2.2 네비게이션 & 헤더**

* 배경: 완전 투명 (black 배경에 겹침)

* 로고: 왼쪽 상단 \- 흰색 왕관 형태 아이콘 (SVG, 독자 제작)

* 메뉴: 오른쪽 상단 햄버거 아이콘 (≡) \- 클릭 시 전체 화면 오버레이 메뉴

* 스크롤 시: 배경이 반투명 검정으로 변경 (backdrop-filter: blur)

* Sticky: position: fixed, z-index: 1000

## **2.3 하단 Sticky CTA 배너**

* 위치: 화면 하단 고정 (position: fixed, bottom: 0\)

* 배경: Vivid Purple (\#8B00FF) 그라디언트

* 왼쪽 텍스트: '내 사업에 맞춘' (줄바꿈) '홈페이지 전략, 솔루션 받기' \- 흰색, Bold

* 오른쪽 버튼: '신청하기' \- 밝은 보라 배경, 흰색 텍스트, border-radius: 8px

* 전체 배너 클릭 시 신청 폼으로 이동

* z-index: 999로 항상 최상위

# **4\. 콘텐츠 전략 (Content Strategy)**

## **4.1 원본 카피라이팅 목록**

아래는 스크린샷에서 추출한 모든 핵심 카피입니다. 동일 구현 시 그대로 사용하거나 업종에 맞게 수정합니다.

### **Hero / 광고 섹션**

* 떠나는 홈페이지와 머무는 홈페이지의 차이

* 보고 싶어지는 기획과 재미있는 AI콘텐츠에서 시작됩니다.

* 고객이 머물지 않는 홈페이지는 매출로 이어지지 않습니다.

* AI 콘텐츠와 기획력으로 '읽게–믿게–행동하게' 만드는 홈페이지

### **랜딩 Hero 섹션**

* Nofail design for you

* 당신을 위한 실패없는 디자인

* \[하단 CTA 배너\] 내 사업에 맞춘 홈페이지 전략, 솔루션 받기 → 신청하기

### **실적 증명 섹션**

* 고객님들이 직접 들려주는 변화 이야기

* 프로젝트 1,100+  |  유지보수 362+  |  고객 만족도 4.9

### **포트폴리오 섹션**

* WEBSITE PORTFOLIO

* 브랜드 쇼핑몰 \- 계림당 : 프리미엄 탈모 제품 브랜드

* AI CONTENT PORTFOLIO

### **홈페이지를 만들려는 이유**

* 안정적인 기업처럼 보이고 싶다

* 거래처에게 신뢰를 주고 싶다

* 많은 고객을 확보하고 싶다

* 경쟁사보다 차별됐으면 좋겠다

* 이 많은 이유들은 하나의 꼭지점에서 시작됩니다

### **매출 공식 섹션**

* 내 회사의 성장 → 매출

* 매출 \= 유입 x 전환

* 노출 → 유입 → 전환

* 홈페이지의 역할은 광고 퍼널 중 전환에 속하죠

* 많은 돈을 써서 유입량을 늘려도 홈페이지가 제 역할을 못한다면 지속적인 손실을 보게 됩니다.

* 즉, 홈페이지 제작은 소비가 아닌 투자로 봐야합니다.

### **고객 태도 분석 섹션**

* 아이러니 하게도 홈페이지에 방문한 고객의 태도입니다.

* 읽지 않는다. / 믿지 않는다. / 행동하지 않는다.

* 이 3가지를 해결해야지만 전환이 잘되는 홈페이지가 됩니다.

* 성공적인 투자가 될 홈페이지는 어떻게 만들어져야할까요?

### **3가지 원칙 섹션**

* \[첫번째\] 읽고 싶게 한다

* 가장 많이 하는 실수는 좋은 걸 다 보여주고 싶은 마음에 홈페이지에 너무 많은 정보를 나열합니다.

* 하지만 정보 과잉은 오히려 가독성과 설득력을 떨어뜨리게 됩니다.

* 잘 읽게 만들려면 불필요한 내용을 과감히 덜어내야 합니다.

* 고객이 궁금해 할 정보만 보여주고 후킹 멘트를 넣어 계속 읽을 수 있도록 유도하고 잘 읽히도록 가독성 있게 정리해야 합니다.

* \[두번째\] 믿을 수 있게 한다

* 고객은 직접 경험하기 전까지 눈앞에 보이는 '겉모습'과 '증거'로만 판단할 수밖에 없습니다.

* 디자인 퀄리티가 낮고 근거 자료가 부족한 홈페이지는 신뢰를 얻지 못합니다.

* 고객이 믿게 하려면 브랜드와 어울리는 깔끔한 디자인과 실제 후기와 사례, 인증서나 현장 사진과 같은 근거가 더해져야 설득력이 완성됩니다.

* \[세번째\] 행동하게 한다

* 좋은 글과 좋은 디자인만으로 고객은 알아서 신청하지 않습니다.

* 신청하는 선택 자체가 부담이 되기 때문에 미루려고 하죠. 행동으로 이어지지 않으면 전환은 일어나지 않습니다.

* 고객이 행동하게 하려면 CTA버튼은 눈 띄는 곳마다 넣어 신청을 쉽게 만들고 놓기지 싫은 심리를 자극하고 실패에 대한 두려움을 줄여줘야합니다.

### **마무리 섹션**

* 이 3가지 원칙은 대표님 회사의 비즈니스 모델을 이해하고 고객을 명확하게 아는 것에서 시작되어야합니다.

* 하지만 이런 기획은 일반적인 기술자들에게는 쉽지 않은 일입니다.

* 요리를 잘하는 것과 장사를 잘하는 게 다른 것처럼 디자인 기술과 실제로 매출을 만들어내는 기획은 전혀 별개의 일입니다.
