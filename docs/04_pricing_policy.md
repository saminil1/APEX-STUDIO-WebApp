## 1. 시장 조사 요약

| 반응형 기업 홈페이지 | 300만~800만 원 |

## 2. APEX STUDIO 가격 정책 (안)

### 포지셔닝 전략

- **시장 대비 중상위 가격대** (상위 30% 포지셔닝)
- 차별화 포인트: AI 콘텐츠 + 전환 중심 설계 + 40년 IT 기획력
- "저가형 제작소"가 아닌 **"매출을 만드는 에이전시"**로 프리미엄 포지셔닝
- 결과 기반 가치 제안 (단순 제작이 아닌 매출 전환 설계)

---

### A. 웹사이트 제작 패키지

| 패키지 | STARTER | STANDARD | PREMIUM | ENTERPRISE |
|---|---|---|---|---|
| **대상** | 소규모/개인사업자 | 중소기업 | 중견기업/브랜드 | 대기업/플랫폼 |
| **가격** | **250만 원~** | **550만 원~** | **1,200만 원~** | **별도 협의** |
| **페이지 수** | 원페이지~5P | 5P~10P | 10P~20P | 제한 없음 |
| **반응형** | O | O | O | O |
| **디자인** | 템플릿 기반 커스텀 | 100% 맞춤 디자인 | 프리미엄 맞춤 디자인 | 브랜드 가이드 연동 |
| **AI 콘텐츠** | 기본 카피 | 카피 + 이미지 5장 | 카피 + 이미지 20장 + 영상 | 풀 콘텐츠 제작 |
| **SEO 최적화** | 기본 SEO | 고급 SEO + GA4 | SEO + GA4 + 전환 분석 | 풀 마케팅 연동 |
| **문의폼/CTA** | O | O (고급) | O (전환 최적화) | O (CRM 연동) |
| **제작 기간** | 2~3주 | 3~5주 | 5~8주 | 8주~ |
| **무상 유지보수** | 1개월 | 3개월 | 6개월 | 12개월 |
| **수정 횟수** | 2회 | 5회 | 무제한 (기간 내) | 무제한 |

### B. 랜딩페이지 제작

| 구분 | BASIC | ADVANCED |
|---|---|---|
| **가격** | **80만 원~** | **200만 원~** |
| **섹션 수** | 3~5 섹션 | 5~10 섹션 |
| **특징** | 반응형 + 기본 애니메이션 | 스크롤 애니메이션 + 전환 설계 + AI 카피 |
| **전환 요소** | 문의폼 1개 | CTA 다중 배치 + A/B 테스트 |
| **제작 기간** | 1~2주 | 2~3주 |

---

### C. 앱 개발 패키지

| 패키지 | MVP | STANDARD | FULL |
|---|---|---|---|
| **대상** | 스타트업/아이디어 검증 | 중소기업 서비스 | 본격 서비스 런칭 |
| **가격** | **800만 원~** | **2,500만 원~** | **5,000만 원~** |
| **플랫폼** | 단일 (iOS 또는 Android) | 크로스플랫폼 (Flutter/RN) | 네이티브 양 플랫폼 |
| **화면 수** | 5~10개 | 10~25개 | 25개~ |
| **백엔드** | Firebase/Supabase | 커스텀 API | 풀 백엔드 |
| **디자인** | 기본 UI 키트 | 맞춤 UI/UX | 프리미엄 UI/UX + 프로토타입 |
| **기능** | 기본 CRUD + 인증 | 결제/푸시/채팅 등 | AI/추천/분석 등 고급 기능 |
| **제작 기간** | 4~6주 | 8~12주 | 12주~ |
| **무상 유지보수** | 1개월 | 3개월 | 6개월 |

## 3. 결제 정책

| 항목 | 내용 |
|---|---|
| **계약금** | 총 금액의 40% (착수 시) |
| **중도금** | 총 금액의 30% (디자인 확정 시) |
| **잔금** | 총 금액의 30% (최종 납품 시) |
| **결제 방식** | 계좌이체, 카드 결제 |
| **세금** | 위 금액은 VAT 별도 |
| **환불 정책** | 착수 전 100%, 기획 단계 70%, 디자인 시작 후 50%, 개발 착수 후 환불 불가 |

## 6. 챗봇 FAQ 반영 가이드

챗봇의 가격 관련 FAQ 답변은 구체적 금액을 명시하지 않고 **"문의폼/상담을 통해 맞춤 견적을 안내"**하는 방식으로 유도합니다.

이유:
- 프로젝트마다 요구사항이 다르므로 일률적 가격 제시가 오해를 유발할 수 있음
- 상담을 통한 리드 수집이 비즈니스 목적에 부합
- 단가표는 내부 영업 기준으로 활용
